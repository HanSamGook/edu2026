복소수의 정의와 연산 마스터

실수 체계의 확장, 복소수의 세계를 탐험해 봅시다.

홈

1. 복소수의 정의

"복소수는 두 실수의 순서쌍 (x, y)로 정의하며, 허수 단위 i를 사용하여 z = x + iy 꼴로 나타냅니다."

  • x
    실수 부분 (Real Part) 기존 실수 체계와 연결되는 지점입니다.
  • y
    허수 부분 (Imaginary Part) i와 결합하여 새로운 수 체계를 형성합니다.

2. 핵심 연산 규칙

덧셈 규칙

실수는 실수끼리, 허수는 허수끼리 끼리끼리 더합니다.

(x₁ + iy₁) + (x₂ + iy₂)
= (x₁ + x₂) + i(y₁ + y₂)

곱셈 규칙

다항식처럼 전개하되, i² = -1을 꼭 적용하세요.

(x₁ + iy₁)(x₂ + iy₂)
= (x₁x₂ - y₁y₂) + i(x₁y₂ + x₂y₁)
